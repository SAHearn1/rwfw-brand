# RootWork Framework Brand Assets

Global CDN for RootWork brand PNGs. Served via Vercel with CORS and immutable cache headers.

## Assets

| File | Description |
|------|-------------|
| `rwfw-seal.png` | Primary RootWork Framework seal |
| `5r-root.png` | 5R phase: Root |
| `5r-regulate.png` | 5R phase: Regulate |
| `5r-reflect.png` | 5R phase: Reflect |
| `5r-restore.png` | 5R phase: Restore |
| `5r-reconnect.png` | 5R phase: Reconnect |
| `favicon.png` | Browser favicon |

## Usage

```html
<img src="https://rwfw-brand.vercel.app/rwfw-seal.png" alt="RootWork Framework" />
```

All assets: `https://rwfw-brand.vercel.app/[filename]`
